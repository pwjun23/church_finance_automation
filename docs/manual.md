# Church Finance Automation - Installation Guide

이 가이드는 구글 스프레드시트에 자동화 시스템을 설치하는 방법을 설명합니다.

## 1. 구글 스프레드시트 준비 (Prepare Sheet)
1. 새로운 구글 스프레드시트를 생성합니다 (이름: `교회재정관리_2026`).
2. 상단 메뉴에서 `확장 프로그램` > `Apps Script`를 클릭합니다.

## 2. 스크립트 복사 (Copy Scripts)
생성된 Apps Script 프로젝트에 아래 파일들의 내용을 각각 복사해서 파일을 만드세요.
(기존의 `Code.gs`는 이름을 바꾸거나 내용을 덮어쓰세요)

- **`Code.gs`**: 메뉴 생성 및 초기 설정
- **`BankImporter.gs`**: 은행 자료 처리 로직
- **`DataEntry.gs`**: 수기 입력 로직
- **`Reporting.gs`**: 보고서 생성 로직

## 3. 초기 설정 (Initial Setup)
1. 스크립트 에디터에서 `Code.gs`를 열고 저장(Ctrl+S)합니다.
2. 스프레드시트로 돌아와서 F5(새로고침)를 누릅니다.
3. 몇 초 후 상단 메뉴바 오른쪽 끝에 **`⛪ 재정 관리 (Finance)`** 메뉴가 나타납니다.
4. **`⛪ 재정 관리` > `1. 초기 설정 (Initialize)`**을 클릭합니다.
   - *권한 요청 창이 뜨면 '계속' -> 계정 선택 -> '고급' -> '... 안전하지 않음(으)로 이동' -> '허용'을 순서대로 눌러주세요.*
   - 완료되면 6개의 시트(`Dashboard`, `Input_Bank`, `Data_General` 등)가 자동으로 생성됩니다.

## 4. 사용 방법 (How to Use)

### A. 은행 자료 가져오기 (Bank Import)
1. 하나은행 엑셀 파일을 엽니다.
2. 데이터를 복사(Ctrl+C)하여 구글 시트 **`Input_Bank`** 탭의 **A1** 셀에 붙여넣기(Ctrl+V) 합니다.
3. 메뉴에서 `2. 은행 자료 처리`를 클릭합니다.
4. 자료가 자동으로 정리되어 `Data_General` 탭으로 이동합니다.

### B. 수기 입력 (Manual Entry)
1. **`Input_Manual`** 탭으로 이동합니다.
2. 입력 폼(B2:B7)에 헌금 내용을 입력합니다.
3. 메뉴에서 `3. 수기 입력 저장`을 클릭합니다. (또는 버튼을 만들어서 이 함수를 연결하면 편합니다)

### C. 보고서 출력 (Report)
1. **`Report_Print`** 탭에서 날짜(B1, B2)가 맞는지 확인합니다.
2. 메뉴에서 `4. 보고서 생성`을 클릭합니다.
3. 생성된 표를 확인하고 인쇄(Ctrl+P) 합니다.

---
**Tip**: 4명의 담당자가 동시에 작업할 때는 `Input_Manual` 시트를 복사해서(예: `Input_Manual_User1`, `Input_Manual_User2`...) 각자 쓰도록 스크립트를 조금 수정하거나, 단순히 돌아가면서 입력하면 충돌을 방지할 수 있습니다.
